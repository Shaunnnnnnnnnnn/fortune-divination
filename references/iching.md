# I Ching Reference

Use I Ching as a symbolic decision framework. Avoid pretending the cast is a literal guarantee.

## Reading Structure

1. 本卦: current structure of the situation.
2. 卦象: use the two trigram images to describe the field.
3. 动爻: active pressure point. If multiple changing lines appear, summarize the shared pattern.
4. 变卦: direction of change.
5. 行动建议: concrete, modest action.

## Trigram Images

- Qian / Heaven: initiative, strength, leadership, creative force.
- Kun / Earth: receptivity, support, patience, nourishment.
- Zhen / Thunder: movement, awakening, shock, first action.
- Xun / Wind/Wood: penetration, gradual influence, adaptation.
- Kan / Water: risk, depth, uncertainty, endurance.
- Li / Fire: clarity, visibility, dependence, insight.
- Gen / Mountain: stillness, boundary, restraint, stopping.
- Dui / Lake: joy, exchange, persuasion, openness.
