# Bazi and Mingli Reference

Use this reference for 八字, 命理, 五行, 事业倾向, 关系模式, and life-theme readings.

## Required Caution

Do not fabricate precise four pillars from a Gregorian birth date unless a reliable calendar conversion implementation is available. Bazi depends on calendar rules, solar terms, time zone, and sometimes true solar time.

If the user provides only birth date/time/place, say:

"我可以先做简化命理倾向分析；如果要严格八字排盘，需要可靠历法换算、节气和出生地时区/真太阳时校正。"

## If Four Pillars Are Provided

Interpret:

- Day master: self-expression and baseline tendency.
- Element balance: support, output, resource, wealth, authority patterns.
- Seasonal context: whether an element is strengthened or weakened.
- Repeated clashes/combinations: recurring tension or adaptation pattern.
- Practical advice: habit, work style, relationship communication, timing awareness.

## Simplified Reading Without Four Pillars

Use:

- User's stated question and current context.
- Birth season as a light metaphor, not as strict calculation.
- Time of day only as a symbolic layer if the user provides it.

Focus on:

- Personality tendencies.
- Relationship needs.
- Career preferences.
- Energy management.
- What to observe and adjust.

Avoid:

- Lifespan, death, illness, pregnancy predictions.
- Exact marriage/divorce dates.
- Claims of guaranteed wealth or failure.
- "Must change name", "must buy object", "must perform ritual".
